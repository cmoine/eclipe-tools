Problem Guide

Errors

  • Missing locale '...': Means that a complete locale file is missing. To fix this right click on the problem(s), choose Quick Fix..., it opens a dialog. It summarizes fixes, click Finish to proceed.
  • Missing key '...' for locale '...':Means that a translation is missing for the given locale/key. Double click to open the editor and fill missing values.

Warnings

  • Key '...' is unused:Means that a translation has be given for a key that does not exist anymore. We advise to remove this key by double clicking on this warning.
